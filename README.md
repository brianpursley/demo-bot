# demo-bot

## Setup

```
make install
```

## Running the Catalog API

```bash
make run-catalog-api
```

Check to make sure the API is running @ http://127.0.0.1:8000/docs

## Running the Email API

```bash
make run-email-api
```

Check to make sure the API is running @ http://127.0.0.1:8001/docs

## Running the Bot

```bash
make run-demo-bot
```
